Understand scoping in Javascript and know to always use const if you can get away with it otherwise let.

Globel scope: variables outside of functions, can be accesed everywhere

function scope: variables inside a function, can be accesed only within that function

Use const by default and use let when you need to reassing a value. Don't use var.

Understand advanced things about functions in JavaScript.

functions are first-class citizens meaning they can be, stored in variables, passed as arguments and returned from other functions

Can write callback functions with different syntax.

Regular function
setTimeout(function() {
  console.log("Hello");
}, 1000);

Arrow function
setTimeout(() => console.log("Hello"), 1000);

Callback function
function greet() {
  console.log("Hello");
}
setTimeout(greet, 1000);

Can create a server with Express from memory.


const express = require('express');
const app = express();

const PORT = 8080;
app.listen(PORT, (error) => {
    if (error) {
        console.log("Error starting the server", error);
        return;
    }
    console.log("Server is running on port", PORT);
});
